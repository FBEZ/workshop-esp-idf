Benvenuto al workshop ESP-IDF avanzato!

Introduzione
#

In questo workshop incontreremo alcuni aspetti più avanzati del framework ESP-IDF, tra cui lo sviluppo modulare tramite componenti, l’event loop, i core dump e le funzionalità di sicurezza.

Tempo stimato: 3 ore.

Agenda
#

Il workshop è suddiviso in quattro parti.

  • Parte 1: Componenti

    • Lezione 1 – Cos’è un componente, come crearlo e come supportare più versioni hardware tramite BSP e configurazioni multiple
    • Esercizio 1.1 – Refactoring del codice creando il componente alarm
    • Esercizio 1.2 – Refactoring del codice creando il componente cloud_manager
    • Esercizio 1.3 – Configurazioni multiple utilizzando sdkconfig

  • Parte 2: Event Loop

    • Lezione 2 – Informazioni base sugli event loop in ESP-IDF, uso degli eventi dei timer e separazione delle responsabilità
    • Esercizio 2.1 – Refactoring del codice per utilizzare l’event loop
    • Esercizio 2.2 – Aggiungere un evento GPIO all’event loop

  • Parte 3: Prestazioni e analisi dei crash

    • Lezione 3 – Size analysis ed uso del core dump per debug
    • Esercizio 3.1 – Analizzare le dimensioni dell’applicazione e suggerire ottimizzazioni
    • Esercizio 3.2 – Analizzare un crash utilizzando i core dump (guidato)
    • Esercizio 3.3 – Analizzare un crash utilizzando i core dump (opzionale)

  • Parte 4: OTA e Funzionalità di Sicurezza

    • Lezione 4 – Fondamenti di OTA, configurazione della tabella delle partizioni, bootloader sicuro, flash cifrata
    • Esercizio 4.1 – Modificare la tabella delle partizioni per supportare OTA
    • Esercizio 4.2 – Utilizzare una tabella delle partizioni personalizzata
    • Esercizio 4.3 – Abilitare la cifratura della flash

Prerequisiti
#

Per seguire questo workshop, assicurati di soddisfare i prerequisiti elencati di seguito.

Software richiesto
#

  • VS Code installato sul proprio computer
  • Estensione ESP-IDF per VS Code aggiunta a VS Code
  • ESP-IDF installato sulla propria macchina
    Può essere installato tramite VS Code oppure utilizzando l’ESP-IDF Installer Manager

Hardware richiesto
#

  • Scheda ESP-C3-DevKit-RUST-1 o ESP-C3-DevKit-RUST-2 (se l’attività è in presenza, la scheda verrà fornita durante il workshop)
    È possibile utilizzare anche una scheda ESP32-C3-DevKit-M/C, ma sarà necessario adattare la configurazione dei pin GPIO di conseguenza.
Se l’evento si svolge in presenza, si consiglia vivamente di installare VS Code e il plugin ESP-IDF prima dell’inizio del workshop. In caso di problemi, sarà comunque previsto un breve momento durante il primo esercizio per completare l’installazione.

Conoscenze base
#

  • Buona conoscenza di:
    • Programmazione in C e del linker
    • Funzioni di callback e puntatori a funzione
    • Protocollo MQTT e il suo utilizzo
  • Programmazione embedded
    • Flashing / Programmazione, linking
    • Familiarità con le periferiche MCU come GPIO e I2C
    • Esperienza di base con ESP-IDF
  • Installazione degli strumenti (VS Code+ estensione ESP-IDF)

Si consiglia vivamente di installare VS Code e il plugin ESP-IDF prima dell’inizio del workshop. Tuttavia, se riscontri problemi, ci sarà un po’ tempo durante il primo esercizio per completare l’installazione.
